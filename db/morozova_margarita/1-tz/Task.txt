Маргарита Морозова. 09-308

Предметная область:
	речные круизы в России.

Постановка задачи:
	Создать единую базу круизов с единым представлением информации, с возможностью бронирования каюты.

Функционал:
	Администратор:
		add owner # Добавить фирму-владельца круиза
	
	Фирма-владелец:
		add cruiseship # Теплоход
		add cabin # Каюта
		add move # Расписание движения теплохода
		add route # Маршрут
		add price # Цена (за каюту на маршруте)
	
	Турист:
		find cruises by: city, dates, routes (маршрут), moves (города на маршруте), prices, cruiseships.
		Бронирование выбранной каюты на выбранном маршруте (автоматически посылается email владельцу)
